2 Entiteiten

2.1 Submission

2.1.1 Class

meb:Submission

2.1.2 Definition

De inzending van een meldingsplichtige publicatie. Een inzending wordt ingestuurd door een (externe) partij en leidt tot een (automatische) verwerking van het gepubliceerde document.

2.1.3 Properties

Name Predicate Range Definition
organization pav:createdBy besluit:Bestuurseenheid Bestuurseenheid die de publicatie gemaakt heeft

href prov:atLocation rdfs:Resource URL waarop het document gepubliceerd is
file nie:hasPart nfo:FileDataObject Bestanden die toegevoegd zijn aan de inzending (door upload of via URL)
