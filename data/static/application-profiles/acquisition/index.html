Luisterpunt Acquisition AP

Laatste aanpassing
2019-09-25

Dit document beschrijft een applicatieprofiel, in dit geval Luisterpunt Acquisition AP.

Samenvatting

Status van dit document

Dit applicatieprofiel dient om het specifieke gebruik van de entiteiten relevant voor de beschreven applicatie te verduidelijken.

Overzicht

In dit document wordt correct gebruik van de volgende entiteiten toegelicht:
| Agent | Annulering | Bronbestand | Externe verwerving | Instantie | Item | Medewerker | Organisatie | Persoon | Productie aanvraag | Productie levering | Titel selectie | Zwartdrukboek bestelling | Zwartdrukboek levering |

Entiteiten

Agent

Beschrijving
Iemand die of iets dat kan handelen of een effect kan teweeg brengen.
Gebruik
Een agent is in het luisterpunt model een organisatie of een persoon.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: contact informatie, naam.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
contact informatie ContactInfo 1 Informatie zoals email, telefoon, adres die toelaat om iemand of iets te contacteren.
naam String 1 Naam waarmee de agent bij voorkeur wordt aangeduid.

Annulering

Beschrijving
Opheffing van een actie, zoals een bestelling.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: beschrijving, creatie datum.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
beschrijving String 1 beschrijving of bijkomende informatie van de annulering.
creatie datum Date 1 datum van annulering

Bronbestand

Beschrijving
Een bronbestand bevat (een deel) van) een creatief werk in een geencodeerde vorm.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: bestandsgrootte, codering, creatie datum, naam, positie, producent, wijzigingsdatum.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
bestandsgrootte QuantativeValue 1 De grootte van het bestand uitgedrukt in een aantal en een eenheid.
codering Format 1 Formaat waarin het werk werd gecodeerd.
creatie datum DateTime 1 Datum waarop het bestand voor het eerst werd weggeschreven.
naam String 1 een beschrijvende naam voor het bestand, eventueel de bestandsnaam.
positie Number 1 De positie van het bronbestand in een reeks van bronbestanden. Duidt de volgorde aan indien een werk bestaat uit meerdere bronbestanden.
producent Organisation 0...1 De orginastie die het bronbestand heeft geproduceerd.
wijzigingsdatum DateTime 1 laatste datum waarop het bestand werd gewijzigd

Externe verwerving

Beschrijving
Verzoek tot verkrijgen van een bronbestand van een toegankelijk werk dat reeds beschikbaar is bij een externe partner.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: creatie datum, publicatie formaat.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
creatie datum dateTime 1 Datum waarop het verzoek werd aangemaakt.
publicatie formaat BookFormatType 1 Het gewenste publicatieformaat van het werk.

Instantie

Beschrijving
Deze entiteit beschrijft een individuele, materiële belichaming van een werk.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: identificator, isbn, luisterpunt id, materiële omvang, publicatie datum, publicatie formaat, speelduur, toegankelijkheids eigenschap, uitgave, vlacc nummer.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
identificator String 1 String gebruikt om het object uniek te identificeren
isbn String 1 ISBN van het boek
luisterpunt id String 1 Identificator die bij luisterpunt aan een titel wordt gegeven.
materiële omvang QuantativeValue 1 Verwijst naar het aantal fysieke onderdelen dat het werk omvat Voorbeeld een quantative value met eenheid 'CD' en aantal '2'
publicatie datum Date 1 Datum waarop voor het eerst openbaar gemaakt. aanwinst datum
publicatie formaat BookFormatType 1 Het publicatieformaat van het boek.
speelduur Duration 0...1 De speelduur van het item (film, audio-opname, evenement, etc.) in ISO 8601-datumformaat.
toegankelijkheids eigenschap AccessibilityFeature 0...* Voorziening in de weergave van het werk in functie van toegankelijkheid. zie https://www.w3.org/wiki/WebSchemas/Accessibility
uitgave String 0...1 Een specifieke uitgave van een boek
vlacc nummer String 0...1 Identificatienummer in de Vlaamse Centrale Catalogus.

Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst

Productie aanvraag

Beschrijving
De bestelling van de productie van een toegankelijk werk bij een producten.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: creatie datum, prioriteit, producent, publicatie formaat, stem voorkeur, werd geïnvalideerd door., Wijzigingsdatum.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
creatie datum DateTime 1 Datum waarop het productieverzoek werd aangemaakt.
prioriteit Code 1 Prioriteit die aan het productieverzoek moet worden gegeven.
producent Organisation 1...* De organisatie bij wie het werk wordt geproduceerd.
publicatie formaat BookFormatType 1 Het gewenste publicatieformaat van het werk.
stem voorkeur Code 1 Man en/of vrouw, Vlaams of Nederlands.
werd geïnvalideerd door. Cancellation 0...1 verwijst naar de annulatie van deze productieaanvraag.
Wijzigingsdatum DateTime 1 Laatste datum waarop de aanvraag werd gewijzigd.

Zwartdrukboek bestelling

Beschrijving
Bestelling of opvraging van een zwartdrukboek.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: aanbieder, besteld artikel, bestelt door, creatie datum, is gebaseerd op, werd geïnvalideerd door.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
aanbieder Agent 0...* De persoon of organisatie bij wie het boek wordt verkregen.
besteld artikel Instance 1...* Verwijst naar het werk dat men met deze bestelling wil verkrijgen.
bestelt door Medewerker De medewerker die de bestelling plaatste.
creatie datum Date 1 Datum waarop het zwartdrukboek werd besteld.
is gebaseerd op WorkSelection 0...1 de werk selectie waar deze bestelling voor werd geplaatst.
werd geïnvalideerd door Cancellation 0...1 verwijst naar de annulatie van de zwart druk bestelling.
